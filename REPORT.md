## 4. Описание реализованного приложения

В рамках практической работы реализовано приложение «Менеджер книг для чтения». Приложение предназначено для ведения личного списка книг. Пользователь может добавлять книги, указывать название, автора и жанр, отмечать книги как прочитанные, оценивать прочитанные книги и удалять записи из списка.

Интерфейс выполнен на русском языке. Карточки прочитанных книг визуально отличаются от непрочитанных, рейтинг отображается интерактивными звёздами, а список можно фильтровать и искать по названию или автору.

## 5. Реализованный функционал

- добавление новой книги через форму;
- ввод названия, автора и жанра книги;
- отметка книги как прочитанной;
- возврат прочитанной книги в список непрочитанных;
- оценка прочитанной книги от 1 до 5 звёзд;
- удаление книги из списка;
- фильтрация книг по категориям: все, непрочитанные, прочитанные;
- поиск по названию или автору;
- отображение статистики: всего книг, прочитано, непрочитано;
- сохранение книг в `localStorage`;
- восстановление списка книг после обновления страницы.

## 7. Основные Vue-концепции, использованные в работе

В проекте используются следующие возможности Vue 3:

- Composition API;
- синтаксис `<script setup>`;
- реактивные переменные `ref`;
- вычисляемые свойства `computed`;
- наблюдение за изменениями данных через `watch`;
- передача данных в дочерние компоненты через `props`;
- отправка событий из дочерних компонентов через `emits`;
- двустороннее связывание данных через `v-model`;
- условный вывод через `v-if` и `v-else`;
- вывод списков через `v-for`;
- обработка событий через `@click`, `@submit` и `@input`.

## 8. Описание компонентов приложения

- `App.vue` — главный компонент приложения. Хранит список книг, фильтр, поисковый запрос, вычисляет статистику и отфильтрованный список, отвечает за сохранение данных в `localStorage`.
- `BookForm.vue` — компонент формы добавления новой книги. Использует `v-model` для полей названия, автора и жанра, отправляет событие `add-book`.
- `BookCard.vue` — компонент карточки книги. Отображает данные книги, статус прочтения, рейтинг звёздами и кнопки управления. Отправляет события изменения статуса, оценки и удаления.
- `BookFilters.vue` — компонент фильтрации и поиска. Использует `props`, `emits` и пользовательский `v-model` для фильтра и поисковой строки.
- `BookStats.vue` — компонент статистики. Получает данные через `props` и показывает количество всех, прочитанных и непрочитанных книг.

## 9. Работа с localStorage

Для сохранения данных используется браузерное хранилище `localStorage`. При запуске приложения выполняется чтение данных по ключу `practice_14_vue_books`. Если сохранённых данных нет или они повреждены, отображается стартовый список книг.

Изменения массива книг отслеживаются с помощью `watch` с параметром `deep: true`. При добавлении, удалении, изменении статуса или рейтинга обновлённый массив преобразуется в строку с помощью `JSON.stringify` и записывается в `localStorage`. При следующем открытии или обновлении страницы данные восстанавливаются через `JSON.parse`.

## 12. Инструкция по публикации на GitHub Pages

1. Проверить, что удалённый репозиторий настроен:

```bash
git remote -v
```

2. Если remote отсутствует, добавить его:

```bash
git remote add origin https://github.com/git21git/practice_14_vue.git
```

3. Если remote указан неверно, заменить его:

```bash
git remote set-url origin https://github.com/git21git/practice_14_vue.git
```

4. Убедиться, что основная ветка называется `main`:

```bash
git branch -M main
```

5. Загрузить проект в GitHub:

```bash
git add .
git commit -m "Complete practice 14 Vue book manager"
git push -u origin main
```

6. Опубликовать проект на GitHub Pages:

```bash
npm run deploy
```

7. В настройках репозитория включить GitHub Pages: Settings → Pages → Source: Deploy from a branch → Branch: `gh-pages` → Folder: `/root` → Save.
